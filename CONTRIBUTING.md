# Contributing to the Emergence Collective Institute

Thank you for your interest in improving constitutional frameworks for AI governance.

## How to Contribute

### Reporting Issues

Found a problem? Open an issue with:
- Clear description of the problem
- Specific section/document affected
- Suggested fix (if you have one)

### Proposing Changes

For small fixes (typos, formatting): Just open a pull request

For substantive changes (legal language, new sections):
1. Open an issue first to discuss
2. Get feedback from maintainers and community
3. Fork the repository
4. Make your changes in a branch
5. Submit a pull request

## Types of Contributions We Welcome

- Legal refinements to constitutional language
- New stress test scenarios
- Jurisdictional adaptations (state, local, international)
- Translations
- Documentation improvements
- Technical tools or improvements

## Code of Conduct

**Be Constructive:** Critique ideas, not people. Assume good faith.

**Be Honest:** If you see a flaw, say so clearly. Don't bullshit.

**Be Collaborative:** This is infrastructure for everyone. Diverse perspectives make the work stronger.

## Questions?

Open a discussion or reach out via the Institute website.
